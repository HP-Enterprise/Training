## 预编译
+ 系统提供的头文件几乎不会发生变化,每次编译C文件都花费了时间
+ 预编译功能使用一个中间文件来加速编译过程

## MVC模式
+ C不是一个面向对象的编程语言
+ 通过创造性的用法,来组织代码
+ 一个简单的MVC模式的使用
